# v0.7.4 Minimal Player Tools

## 基础版本

基于 v0.7.3 继续小改。

## 本次改动

只改播放工具按钮的 UI 和交互，不改播放栏本体、不改歌词、不改数据。

## 调整内容

1. 去掉两个按钮的外部大卡片感。
2. 去掉“随机 / 音量 / 百分比”等按钮文字，只保留图标。
3. 按钮大小改成接近原播放栏左右切换按钮。
4. 按钮颜色加深，降低红色描边，避免突兀。
5. 播放模式点击后不再弹出四选一面板，而是按顺序循环切换：
   - 顺序
   - 循环
   - 单曲
   - 随机
6. 音量点击后只弹出滑轨。
7. 音量面板取消静音按钮和百分比文字。
