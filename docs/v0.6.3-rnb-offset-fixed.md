# v0.6.3 R&B 歌词偏移修正

## 修改内容

- 给 `R&B All Night (Masiwei Remix)` 添加：

```json
"lyricsOffset": 0.45
```

## 原因

测试发现：

- `Babe` 歌词同步基本正常
- `R&B All Night` 整体慢半拍

因此判断是 R&B 这首歌的 LRC 与当前 FLAC 音源存在整体偏移，而不是全局同步逻辑问题。

## 后续调试

如果 R&B 仍然慢，可以继续调大：

```json
"lyricsOffset": 0.6
```

如果提前过头，可以调小：

```json
"lyricsOffset": 0.25
```

如果某首歌单独偏移，优先只改那首歌的 `lyricsOffset`，不要改全局代码。
