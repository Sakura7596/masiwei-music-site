## 歌词同步调试

本版增加：

```json
"lyricsOffset": 0
```

如果歌词慢，可以把该值调成正数，例如：

```json
"lyricsOffset": 0.35
```

如果歌词快，可以改成负数，例如：

```json
"lyricsOffset": -0.25
```

另外，歌词滚动动画从 0.46s 缩短到 0.20s，高亮动画缩短到 0.16s，减少“慢半拍”的视觉感。
