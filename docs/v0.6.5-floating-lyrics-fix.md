# v0.6.5 Floating Lyrics Fix

## 修复内容

1. 未播放时左侧完全空白，不显示 lyricPreview 或提示文案。
2. 播放中切换到其他专辑时，左侧继续显示正在播放歌曲的歌词，不再跟随浏览专辑清空。
3. 去掉同步歌词区域的背景、雾化层、mask 和伪元素，改成纯透明浮字。
4. 无同步歌词的歌曲不显示歌词面板，避免出现提示框。
5. 隐藏旧的 `.left-caption`，避免和同步歌词面板叠加。
6. 增强第一次播放时歌词高亮校准。

## 设计规则

- 没播放：左侧空白
- 播放有 LRC 的歌曲：左侧显示浮动同步歌词
- 播放无 LRC 的歌曲：左侧不显示歌词面板
- 浏览别的专辑：歌词仍然保持当前播放歌曲
