# v0.6.1 Sync Lyrics Fix

## 修复内容

v0.6 中 `getTrack()` 只保留了 `title/audio/lyrics`，导致 `lyricsUrl`、`lyricsFormat`、`lyricPreview` 等字段被丢弃，页面会误判为没有同步歌词。

本版修复：

- `getTrack()` 保留完整 track 对象
- 正确读取 `lyricsUrl`
- 正确读取 `lyricPreview`
- 继续支持字符串 tracks 的旧数据格式

## 测试

本地运行：

```bash
python -m http.server 8000
```

打开：

```txt
http://localhost:8000
http://localhost:8000/lyrics/rnb-all-night-masiwei-remix.lrc
```
