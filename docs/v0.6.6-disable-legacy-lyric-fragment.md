# v0.6.6 Disable Legacy Lyric Fragment

## 修复原因

v0.6.5 仍保留旧函数 `renderLyricPanel()`，它会在切换专辑时根据当前专辑的 `lyricPreview` 渲染旧的 `LYRIC FRAGMENT` 大字区域，覆盖/干扰新的同步歌词面板。

## 本版修复

- 将旧 `renderLyricPanel()` 改为 no-op。
- `updateScene()` 不再调用旧歌词片段渲染。
- 未播放时左侧完全空白。
- 播放中切换专辑时，左侧继续保持当前播放歌曲的同步歌词。
- 强制隐藏旧 `.lyric-line` / `.lyric-note`。
- 同步歌词渲染前移除旧 `.show` 状态。

## 预期效果

1. 未播放时，R&B 页面左侧不会出现“我们不用睡觉”。
2. 播放 R&B 或 Babe 后，左侧显示同步滚动歌词。
3. 播放中切换到其他专辑，歌词继续显示当前播放歌曲。
4. 不再出现旧的 lyric fragment 文本框或大字歌词。
