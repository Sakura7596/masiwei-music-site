# v0.6 Sync Lyrics Notes

## 新增功能

- 左侧同步滚动歌词面板
- 支持读取 `track.lyricsUrl`
- 支持 LRC 时间轴解析
- 播放时按 `audio.currentTime` 高亮当前歌词
- 当前歌词居中滚动
- 相邻歌词弱高亮
- 无歌词时显示提示
- 未播放时显示当前专辑/歌曲的 lyricPreview

## 数据要求

每首歌建议写：

```json
{
  "title": "歌曲名",
  "audio": "https://R2/audio/song.flac",
  "lyricPreview": "一句歌词",
  "lyricsUrl": "./lyrics/song.lrc",
  "lyricsFormat": "lrc",
  "lyrics": ""
}
```

## 文件位置

```txt
lyrics/song.lrc
```
