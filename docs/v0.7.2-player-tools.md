# v0.7.2 Player Tools

## 基础版本

基于用户上传的 `v0.7.1.zip` 稳定版修改。

## 改动范围

不改播放栏本体，只在底部播放栏右侧新增两个按钮：

1. 播放模式
2. 音量

## 播放模式按钮

常驻显示一个按钮，点击后展开模式菜单：

- 顺序：按当前专辑顺序播放，最后一首结束后停止
- 循环：当前专辑列表循环播放
- 单曲：当前歌曲循环
- 随机：当前专辑内随机播放，尽量避免连续同一首

播放模式保存到：

```txt
localStorage: msw-play-mode
```

## 音量按钮

常驻显示一个按钮，点击后展开音量面板：

- 0% - 100% 滑杆
- 静音 / 取消静音
- 音量百分比显示在按钮上

音量保存到：

```txt
localStorage: msw-volume
localStorage: msw-muted
```

## UI 风格

按用户提供截图模仿：

- 深色半透明圆角卡片
- 内部圆形图标
- 中文标签
- hover 微亮
- active 红色描边
- 不覆盖专辑列表，不塞进播放栏结构内部
