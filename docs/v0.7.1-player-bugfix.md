# v0.7.1 Player Bugfix

## 修复内容

1. 删除旧的“未绑定 MP3，正在播放占位音色”文案。
2. 音频 URL 存在但播放失败时，不再退回占位音色，避免旧状态污染播放器。
3. 增加 `playRequestToken`，防止上一首歌的异步失败回调覆盖当前正在播放的歌曲 UI。
4. 修复抽屉点击歌曲时使用 `currentAlbum` 的问题。现在每个曲目按钮携带 `data-play-album`，点击会播放当前浏览专辑的对应歌曲。
5. 修复播放状态只按 track index 判断的问题。现在按 album + track 同时判断。
6. 歌词加载增加 `encodeURI` 备用路径，对中文歌词路径更稳。
7. `timeupdate` 中直接更新歌词高亮，减少第一次播放/切歌时歌词不刷新的情况。
8. 抽屉中 MP3 文案改为通用 `AUDIO READY / NO AUDIO`，避免 FLAC 场景下误导。

## 重点测试

- 黑马王子专辑中任意几首歌的歌词是否正常显示。
- 快速切歌时播放器是否还会出现旧的“未绑定 MP3”字样。
- 打开某张专辑抽屉后点击歌曲，是否播放该专辑的正确歌曲。
- 播放中切换到其他专辑，左侧歌词是否继续显示当前播放歌曲。
