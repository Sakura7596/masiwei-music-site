# v0.4.1 Preload Notes

## 修复的问题

v0.4 把封面从 JSON 里拆成外部 WebP 后，首屏会先渲染页面再加载图片，网络稍慢时会出现封面短暂空白。

## 本版优化

- 增加首屏 loader。
- 页面初始化时优先预加载当前专辑和相邻专辑封面。
- 首屏封面加载后再隐藏 loader。
- 剩余封面后台预加载。
- 封面图片增加淡入效果，避免突然闪现。
- 切换专辑时主动预加载邻近封面。
- 保持 R2 音频方案不变。
